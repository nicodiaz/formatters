Archivos de Formateos de Codigo
===============================

Se almacenan los archivos necesarios para formatear codigos de distintos
lenguajes, frameworks y plataformas. 

Son de propia autoria, y ajustados a distintos proyectos. No siguen ningun
standard. 

Cualquier sugerencia es bienvenida!

PHP 
====

Se trabaja a partir del standard PSR-2

[Referencia]: http://www.php-fig.org/psr/psr-2/
